# Clean-mtab-config
A simple and clean configuration file to use for the 'mtab' chrome extension.

How to use:
1. Install the extension mtab: https://chromewebstore.google.com/detail/mtab/fdaphilojaklgkoocegabckfanjoacjg
   <img width="409" height="203" alt="Screenshot_20260111_185559" src="https://github.com/user-attachments/assets/21212969-bd18-4bfa-9ea6-5f7fa2cd001d" />

2. On the home page of your browser, click on the settings icon at the bottom of the tab.
   <img width="1913" height="910" alt="Screenshot_20260111_190008-1" src="https://github.com/user-attachments/assets/9060978d-7d96-4c51-9439-8a7ebc6f52af" />
3. Scroll a little to uncover the menu, where it gives you the option to save or import and export and so on, and click on the import and save option.
4. There you can paste the text from the Mtab-save text file in the repo.
5. Click on ok and enjoy!

6. There is also a .json file in the repo, be sure to check that out!
